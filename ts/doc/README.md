This is a meta-package to consolidate VoidMerge typescript documentation.

## Writing a VoidMerge Application

If you are looking for API documentation for writing a VoidMerge application:

- You want the `@voidmerge/voidmerge-code` package.
- API Docs: {@link VoidMergeCode}.

## Interacting with a Running VoidMerge Server Instance

If you are looking for client tools for interacting with a running VoidMerge server instance:

- You want the `@voidmerge/voidmerge-client` package.
- API Docs: {@link VoidMergeClient}.
